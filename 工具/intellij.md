| ------ | :-------------- | ------------------------------------------------ |
| 导航   |                 |                                                  |
|        | 用标签来回切换  | 使用F3添加标签，cmd+2 打开标签窗口，用于快速定位 |
|        |                 |                                                  |
|        |                 |                                                  |
|        |                 |                                                  |
|        |                 |                                                  |
| 编辑   |                 |                                                  |
|        | 修复错误        | 编辑区内，F2， shift+F2，快速定位下、上一个错误  |
|        |                 |                                                  |
|        |                 |                                                  |
|        |                 |                                                  |
|        |                 |                                                  |
| 自定义 |                 |                                                  |
|        | cmd + shift + s | select opened file                               |
|        |                 |                                                  |
